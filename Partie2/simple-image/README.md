# Microservices

## Sommaire
1. Serveur
2. Client

Afin de lancer les scripts, utiliser python 3. Par exemple :
```python3 server.py```

Si besoin installer la dépendance :
* ```pip3 install jsonpickle```
* ```pip3 install flask```

Ces scripts permettent de transférer une image en HTTP d'un client au serveur.
